Observable - Observable Data and Configuration

The data and configuration passed to Durandal Grid can be observable, and changes will cause the grid to update.

Example

Javascript

/* Create observables /*

return {
    gridConfig: { 
        data: observableData,
        pageSize: pageSize,
        showPageSizeOptions: showPageSizeOptions,
        pageSizeOptions: pageSizeOptions,
        columns: columns
    }
};

HTML

<table class="paging-container grid-table" data-bind="grid: gridConfig"></table>
